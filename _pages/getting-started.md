Prerequisites to getting your chromebook to install and run games from gog.com.

## Install Linux for Chromebook (Crostini)

If your chromebook supports linux, you can install it from the chromebook settings.  Once installed, a icon for the terminal application will appear in the application launcher.

### Upgrades

Keeping the linux system updated requires you to manually run the upgrade commands

    sudo apt-get update
    sudo apt-get upgrade

### innoextract

A tool to unpack installers generated by Inno Setup.  Setup files for windows games on GOG.com are all built with Inno Setup.  This application is needed to extract the game data from gog installers.
Because GOG.com updates the game installers on a regular basis with the latest version of Inno Setup, it is necessary to install and keep the latest version of innoextract for compatibility.
Run this command to add the latest builds of innoextract to the linux repository

    sudo echo "deb http://download.opensuse.org/repositories/home:dscharrer/Debian_7.0/ /" >> /etc/apt/sources.list

This second command will install the GPG key on 

    wget http://download.opensuse.org/repositories/home:dscharrer/Debian_7.0/Release.key | sudo apt-key add Release.key
